# xiaomi-lidar-lds01rr
Interface Winforms for lidar xiaomi LDS01RR or LDS02RR and ESP32 (arduino)

arduino ESP32 (15 pins):
* library: https://github.com/kaiaai/LDS

components:
* transistor BD137 (NPN)
* resistor 1K ohm
